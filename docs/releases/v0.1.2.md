# nix-agent v0.1.2

## Fixes

- Add `nixpkgs-fmt` to the packaged `nix-agent` runtime path.
- Fix end-to-end MCP workflows that call `run_formatters` from an installed `nix-agent` binary.

## Why this release matters

`v0.1.1` started correctly, but installed workflows could still fail at `run_formatters` because `nixpkgs-fmt` was not available on `PATH` inside the packaged runtime.

## Verification

```bash
pytest
nix build .#default
```
