# nix-agent v0.1.1

## Fixes

- Fix `nix-agent` startup with current FastMCP releases.
- Replace private FastMCP component access with explicit tool registration.

## Why this release matters

`v0.1.0` could be installed successfully but the `nix-agent` binary crashed on startup in the packaged runtime. This patch release fixes that MCP startup failure.

## Verification

```bash
pytest
nix build .#default
```
