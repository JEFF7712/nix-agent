# nix-agent v0.1.3

## Fixes

- Resolve `nixos-rebuild` to its absolute path before calling it through `sudo`.
- Make passwordless sudo automation work with narrow sudoers allowlists that match the real binary path.
- Add clearer privileged-automation guidance for hosts using `nix-agent` non-interactively.

## Why this release matters

`v0.1.2` fixed formatter availability, but privileged automation could still fail if the sudoers rule matched the fully resolved `nixos-rebuild` path while `nix-agent` invoked `sudo nixos-rebuild ...` with a bare command name.
